## What changed

This PR adds CSV export for the company skills taxonomy.

### Client (ango-scholars-client)

- New API hook `useExportSkillsTaxonomy` calling `GET /skills/export-csv`
- `SkillsTable` toolbar: Export button with download via blob response
- i18n keys for export label and error toasts

### Server (ango-scholars-server)

- New controller handler `GET /skills/export-csv` returning `text/csv`
- Requires company member with `MANAGE_SKILLS` action
- CSV columns: category, mainDiscipline, secondaryDiscipline
