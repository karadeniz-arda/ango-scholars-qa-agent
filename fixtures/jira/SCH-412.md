## Summary

Company admin can export skills taxonomy as CSV from Skills page

## Description

As a company admin, I want to download our skills taxonomy as a CSV file so I can share it with HR or import it into external tools.

### Acceptance criteria

- On `/company/skills`, an Export (or Download CSV) action is visible to users with `MANAGE_SKILLS` permission.
- Clicking Export triggers `GET /skills/export-csv` and downloads a CSV file with column headers: category, mainDiscipline, secondaryDiscipline (or equivalent).
- Talent users and unauthenticated users cannot access the export endpoint.
- Export shows a success toast or browser download; errors show a user-visible message (not a silent failure).
